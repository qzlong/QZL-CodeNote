##### 53.最大子序和
> 给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
****
##### 解法一：动态规划
> 思路：令F(n)为以nums[n]为序列尾的子序列的和的最大值，则F(n)=max(F(n-1)+nums[n],nums[n])
```c++
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int res = nums[0];
        int f   = nums[0];
        for(int i=1;i<nums.size();++i){
            f = max(f+nums[i],nums[i]);
            res = max(f,res);
        }
        return res;
    }
};
```
