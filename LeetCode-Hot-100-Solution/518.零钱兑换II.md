### 518.零钱兑换II
> 给定不同面额和一个总金额。写出函数来计算可以凑成总金额的硬币组合数。假设每一种面额的硬币有无限个。 
***
##### 解法一：动态规划
> 属于完全背包问题的变种，具体参见背包九讲
```c++
class Solution {
public:
    int change(int amount, vector<int>& coins) {
        vector<int> dp(amount+1,0);
        dp[0]=1;
        for(int coin:coins)
            for(int i=coin;i<=amount;i++)
                dp[i]+=dp[i-coin];
        return dp[amount];
    }
};
```
