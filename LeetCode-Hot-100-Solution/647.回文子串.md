### 647.回文子串
> 给定一个字符串，你的任务是计算这个字符串中有多少个回文子串。
> 具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。
***
##### 解法一：中心拓展法
```c++
class Solution {
public:
    int countSubstrings(string s) {
        int n=s.size(),ans=0;
        for(int i=0;i<n;++i)
        {
            for(int j=0;j<=1;++j){
                int l=i;
                int r=i+j;
                while(l>=0&&r<n&&s[l--]==s[r++])
                    ans++;
            }
        }
        return ans;
    }
};
```
